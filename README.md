# 语录收集工具 · Screenshot to MD

一个 Electron 桌面应用，用来把手机截图里的短语、语录、摘抄整理成 Markdown 文档。

## 当前流程

1. 添加或拖入手机截图。
2. 应用先用本地 OCR 提取每张截图中的文字。
3. 再把 OCR 文本发送给 OpenAI 兼容的文本模型 API。
4. 模型负责清理 UI 噪音、合并连续截图、去重，并输出纯 Markdown。

这样比直接把截图交给多模态模型更稳定，也更容易检查中间结果。

## 启动

```bash
npm start
```

## 配置

在应用右上角设置中填写：

- API Key
- API 地址，例如 `api.hunyuan.cloud.tencent.com`
- 模型名称，可以填写你账号可用的 OpenAI 兼容文本模型

OCR 使用本地 `tesseract.js`，已内置简体中文和英文语言包。

## 整理规则

- 忽略按钮、导航栏、评论、点赞数、时间戳、用户名、话题标签等 UI 噪音。
- 连续截图如果属于同一段内容，会按顺序合并并去掉重复部分。
- 单句语录使用 Markdown 引用格式。
- 长段落按主题或来源使用二级标题分组。
- 尽量保留原文，只做必要清洗、合并和断句。
